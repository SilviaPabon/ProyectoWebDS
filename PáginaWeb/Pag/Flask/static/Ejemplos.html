Explicación de cada Paradigma y diferencias de codificación respecto a las Estructuras de Datos

Python

Python es un lenguaje de programación de enfoque multiparadigma, trabaja programación orientada a objetos y procedimental. Algunas de sus particularidades a la hora de trabajar Estructuras de Datos con POO son las siguientes: En Python, una variable es una ubicación de un objeto de cierto tipo, al que se le asocia el nombre de la variable. La variable no guarda nada dentro de ella, excepto referencias (ubicación donde se encuentra realmente almacenado el objeto). Cuando las referencias a un objeto son eliminadas o la variable se elimina, apuntando a None, en Python hay un Recolector de basura automático que cuando las detecta las “recoge”. El this de Java, aquí es self, obligatoriamente va como parámetro de los métodos, aunque no lleve ningún otro. Sólo existe un constructor __init__, en donde deben ir los atributos de la clase, en este no existe la sobreescritura como se conoce en Java. La forma de redacción de código es más sencilla aquí. No es necesaria una interface, y no hay tanta restricción sobre encapsulamiento privado o público [1].

JAVA

Java es un lenguaje de programación reconocido este pertenece al paradigma de programación orientada a objetos, lo cual quiere decir que es próximo a la forma de pensar del humano. Como ya sabemos las estructuras de datos son hechas para hacer que la organización de los ya mencionados datos sea más fácil y la eficiencia de si manipulación sea mucho mejor. Estas estructuras son formadas a través de clases la cuales contienen sus respectivos métodos, los más particulares son buscar, insertar, eliminar, listar, entre otros; estos están compuestos por sus respectivos nodos [3].

C++

C++ es un lenguaje multiparadigma, con posibilidad de trabajar en programación genérica, programación estructurada y programación orientada a objetos. La programación en Java es muy similar a la programación en C++, en el caso del manejo de estructuras de datos, en C++ se permite el uso de punteros, los punteros son una variable que hace referencia a la dirección de memoria donde se guarda el dato especificado. Además, en este lenguaje no basta con eliminar los enlaces entre nodos, también se debe realizar la eliminación del nodo para así cumplir con un correcto método de eliminación[2].
